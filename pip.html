Turn2us PIP helper

Turn2us PIP helper embedded webpage demo.

Introduction:

This is a simple example of an HTML webpage which has the Turn2us PIP Helper embedded into it. This will show you how the Helper will behave on your website. Please scroll down the page to see the embedded demo.



Implementation:

If you would like to add the embedded Turn2us PIP helper to your website, please copy and paste the following code into one of your webpages.

HTML code:


<iframe id="pip-iframe" src="https://uat-pip.turn2us.org.uk/how-do-you-start-your-claim/?portal=true&" width="100%" height="640" scrolling="no" frameborder="0">
  <p>Your browser does not support iframes.</p>
</iframe>
<script src="https://uat-pip.turn2us.org.uk/static/js/portal.min.js" crossorigin="anonymous"></script>

Note: When copy and pasting HTML code please many sure you don't accidently encode the text and do not introduce line breaks.
