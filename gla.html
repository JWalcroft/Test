Turn2us: Grant Search

Greater London Authority, Grants Search embedded webpage demo.

Introduction:

This is a simple example of an HTML webpage which has the Turn2us Grants Search embedded into it. This will show you how the Grants Search will behave on your website and has been styled to match the GLA brand guide. Please scroll down the page to see the embedded Grant Search.

Implementation:

If you would like to add the embedded Grants Search to your website, please copy and paste the following code into one of your webpages.

HTML code:


<iframe id="gs-iframe" src="https://portal-grants-search.turn2us.org.uk/survey/?portal=true&id=gla_live&utm_source=gla_live&utm_medium=iFrame&utm_campaign=GSPortal" width="100%" height="800" scrolling="no" frameborder="0">
  <p>Your browser does not support iframes.</p>
</iframe>
<script src="https://portal-grants-search.turn2us.org.uk/static/js/portal.min.js" crossorigin="anonymous"></script>

Note: When copy and pasting HTML code please many sure you don't accidently encode the text and do not introduce line breaks.
